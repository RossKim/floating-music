# Floating Music

## Description

- Overview
  You can control music players without going to home, swipe status bar and opening music player apps, by the overlap music control.

- Features
  It is possible to control the 3rd party music player (play, pause, rewind, forward). In addition, it is possible to resize the overlay view by dragging bottom-left or bottom-right of the overlay view to show more controls.
